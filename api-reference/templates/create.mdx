---
title: 'Create template'
description: 'Create a new email template'
---

## Endpoint

```
POST /templates
```

## Request body

<ParamField body="name" type="string" required>
  Template name for internal reference
</ParamField>

<ParamField body="subject" type="string">
  Default subject line (supports variables)
</ParamField>

<ParamField body="html_content" type="string" required>
  HTML template content (supports variables)
</ParamField>

<ParamField body="text_content" type="string">
  Plain text template content
</ParamField>

## Template variables

Use double curly braces for variables in your templates:
- `{{first_name}}` - Subscriber first name
- `{{last_name}}` - Subscriber last name
- `{{email}}` - Subscriber email
- `{{unsubscribe_url}}` - Unsubscribe link
- Custom fields: `{{custom_field_name}}`

## Response

<ResponseField name="data" type="object">
  Created template object
</ResponseField>

## Example request

```bash
curl -X POST https://api.mailist.com/v1/templates \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Welcome Email",
    "subject": "Welcome to {{company_name}}, {{first_name}}!",
    "html_content": "<h1>Welcome {{first_name}}!</h1><p>Thanks for subscribing.</p>"
  }'
```

## Example response

```json
{
  "data": {
    "id": "tmpl_new456",
    "name": "Welcome Email",
    "subject": "Welcome to {{company_name}}, {{first_name}}!",
    "created_at": "2025-12-30T09:00:00Z"
  }
}
```
