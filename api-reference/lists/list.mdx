---
title: 'List mailing lists'
description: 'Retrieve all mailing lists'
---

## Endpoint

```
GET /lists
```

## Query parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="integer" default="50">
  Number of lists per page (max 100)
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of mailing list objects
  
  <Expandable title="List object">
    <ResponseField name="id" type="string">
      Unique list identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      List name
    </ResponseField>
    
    <ResponseField name="description" type="string">
      List description
    </ResponseField>
    
    <ResponseField name="subscriber_count" type="integer">
      Number of active subscribers
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      ISO 8601 timestamp of creation
    </ResponseField>
  </Expandable>
</ResponseField>

## Example request

```bash
curl https://api.mailist.com/v1/lists \
  -H "Authorization: Bearer YOUR_API_KEY"
```
