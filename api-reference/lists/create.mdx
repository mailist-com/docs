---
title: 'Create mailing list'
description: 'Create a new mailing list'
---

## Endpoint

```
POST /lists
```

## Request body

<ParamField body="name" type="string" required>
  List name
</ParamField>

<ParamField body="description" type="string">
  List description
</ParamField>

<ParamField body="from_name" type="string">
  Default sender name for campaigns
</ParamField>

<ParamField body="from_email" type="string">
  Default sender email for campaigns
</ParamField>

## Response

<ResponseField name="data" type="object">
  Created mailing list object
</ResponseField>

## Example request

```bash
curl -X POST https://api.mailist.com/v1/lists \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Product Updates",
    "description": "Subscribers interested in product announcements",
    "from_name": "Product Team",
    "from_email": "product@mailist.com"
  }'
```

## Example response

```json
{
  "data": {
    "id": "list_new789",
    "name": "Product Updates",
    "description": "Subscribers interested in product announcements",
    "subscriber_count": 0,
    "created_at": "2025-12-30T09:00:00Z"
  }
}
```
