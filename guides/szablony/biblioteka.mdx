## Użycie szablonu

<Steps>
  <Step title="Przeglądaj bibliotekę">
    **Szablony** → **Biblioteka** → Filtruj według kategorii
  </Step>

  <Step title="Podgląd">
    Kliknij szablon aby zobaczyć preview na desktop i mobile
  </Step>

  <Step title="Użyj szablonu">
    **Użyj tego szablonu** → Dostosuj treść i kolory → Zapisz
  </Step>
</Steps>

## Popularne szablony

### Newsletter firmowy

```
Sekcje:
- Header z logo
- Hero image z głównym komunikatem
- 3 kolumny z artykułami
- CTA główny
- Footer z social media i unsubscribe
```

**Użycie:** Cotygodniowy update dla subskrybentów

### Welcome Email

```
Sekcje:
- Osobiste powitanie: "Witaj {{firstName}}!"
- Co dalej? (3 kroki)
- Pomocne linki
- Kontakt do supportu
```

**Użycie:** Pierwszy email po zapisie na listę

### Flash Sale

```
Sekcje:
- Countdown timer
- Duży rabat: "-50%"
- Najpopularniejsze produkty (3-4)
- Wyraźny CTA: "Kup teraz"
- Urgency: "Tylko dziś!"
```

**Użycie:** Limitowane czasowo promocje

## Customizacja

Każdy szablon możesz dostosować:

- **Kolory**: Zmień na kolory swojej marki
- **Logo**: Dodaj swoje logo
- **Treść**: Edytuj tekst i obrazki
- **Layout**: Dodaj/usuń sekcje

<Tip>
  **Pro tip:** Zapisz dostosowany szablon jako własny, aby używać go wielokrotnie.
</Tip>
